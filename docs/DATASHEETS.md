# STM32F103 数据手册和参考资料

本文档提供 STM32F103C8T6 相关的中英文手册获取途径。

## 📚 官方中文手册

### 1. STM32F103 中文数据手册

**文件名**: STM32F103x8/xB 中文数据手册

**内容**:
- 芯片概述和特性
- 引脚定义和封装
- 电气特性
- 时序参数
- 存储器映射

**获取途径**:

1. **ST 中文官网**
   - 网址: https://www.st.com/zh
   - 搜索: STM32F103C8
   - 点击产品页面 → 资源 → 数据手册
   - 选择"简体中文"版本

2. **直接下载链接**
   ```
   产品型号: STM32F103C8T6
   文档编号: DS5319 (中文版)
   文件名: stm32f103c8t6_cn.pdf 或类似
   ```

3. **国内镜像站**
   - 21IC 电子技术论坛：https://www.21ic.com/
   - 电子发烧友：https://www.elecfans.com/
   - 硬禾学堂：https://www.yingsheng.com/

## 📖 推荐学习路径

### 初学者（第1-2周）

1. **先看**: STM32F103 中文数据手册
   - 了解芯片基本特性
   - 熟悉引脚定义
   - 理解最小系统

2. **然后看**: 参考手册第一部分
   - 存储器映射
   - RCC 时钟配置
   - GPIO 基础

3. **实践**:
   - LED 闪烁（参考 docs/EXAMPLES.md）
   - 按钮输入

### 进阶（第3-4周）

1. **深入学习**: STM32F103 中文参考手册
   - UART 串口通信
   - 定时器配置
   - 中断系统

2. **参考**: Cortex-M3 权威指南
   - 异常处理机制
   - NVIC 中断控制器
   - SysTick 系统定时器

3. **实践**:
   - UART 通信（参考 docs/EXAMPLES.md）
   - 定时器中断

### 高级（第5周+）

1. **专题学习**:
   - ADC 模数转换
   - DMA 直接内存访问
   - 低功耗模式
   - CAN 通信

2. **系统设计**:
   - RTOS 移植
   - 通信协议栈
   - 驱动框架设计
